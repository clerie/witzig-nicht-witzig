# Witzig! - Nicht Witzig!
Dieses Firefox Plugin ersetzt die Daumen in YouTube getreu dem Känguru durch Witzig und Nicht Witzig.

# Installation
1. Herunterladen
2. Entpacken
3. In Firefox nach about:debugging navigieren
4. Add-on temporär laden
5. YouTube aufrufen
